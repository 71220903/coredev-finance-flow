## üì± How to Use

### For Developers
1. **Connect Wallet**: Link your Web3 wallet (MetaMask recommended)
2. **Create Profile**: Import GitHub data and showcase your projects
3. **Create Market**: Set loan terms (amount, interest rate, duration)
4. **Get Funded**: Receive funding from interested lenders
5. **Build & Repay**: Complete your project and repay on schedule

### For Lenders
1. **Connect Wallet**: Link your Web3 wallet with funds
2. **Browse Markets**: Explore available developer lending opportunities
3. **Analyze Risk**: Review trust scores, projects, and terms
4. **Invest**: Deposit funds into promising markets
5. **Earn Returns**: Receive principal + interest upon repayment

### Navigation
- **Dashboard**: Overview of your activity and portfolio
- **Marketplace**: Browse and filter available lending opportunities
- **Analytics**: Detailed market insights and performance metrics
- **Profile**: Manage your developer profile and achievements
- **Settings**: Configure notifications, security, and preferences
